# 3 лабораторная работа по ООП


## Вариант 12
### Описание задания:
Класс «Угол в диапазоне [-2π, 2π]» - может задаваться в радианах и градусах, с возможностью конвертации.

С его помощью можно: 
- определить тип угла, 
- складывать и вычитать углы, 
- сравнивать углы,
- предопределить углы, равные 0 и π.
